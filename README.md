# Is the proxy blocked?
###### More of a Linewize tool
### A nice tool
Checking if proxy links are blocked using brute forcing style.
## How to Use
1. Make sure you have python installed
2. Clone the repo
   ```bash
   git clone https://github.com/Lonely-Corporation/Is-the-Proxy-Blocked.git
   ```
3. ```bash
   cd Is-the-Proxy-Blocked
   ```
4. ```bash
   pip install requests
   ```
4. Edit the links.txt file and fill it with links to check (example links in links.txt)
5. Run the code
   ```bash
   python main.py
